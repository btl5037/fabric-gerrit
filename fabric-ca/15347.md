Prior to this change set, you had to specify lists as
a comma-separated lists without surrounding brackets
as follows:
   a,b,c
This change set allows either of the following two forms:
   a,b,c
   [a,b,c]

For example:
   --tls.certfiles a,b
or
   --tls.certfiles [a,b]
