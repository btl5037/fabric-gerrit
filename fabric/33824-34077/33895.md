etcd/wal pkg hardcodes SegmentFileSize to 64MB. If we set
```
SnapshotIntervalSize = 16MB
MaxSnapshotFiles = 4
```
then we will have no more than 2 segment file.

Note: if block size is larger than 32MB, we would still end up
with more segment files.

This CR also changes SnapshotCatchUpEntries to 4 (previously 20),
to reduce data kept in memory.
