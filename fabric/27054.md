This change set adds an implementation of a deliver client
for intra-cluster block replication.

The deliver client has 2 methods:

1) Close() - Closes the connections to the ordering node if applicable.
2) PullBlock(seq uint64) *common.Block - This method makes the deliver
   client connect to some ordering node that has the given sequence,
   and fetch this block from it.
   It blocks until a block with the given sequence can be retrieved.

This change set also changes the Gopkg.lock because
I dep ensure requires it due to imports
