This change set adds a gRPC service that services requests from clients
that are related to peer resource tree configuration.

The service has 2 API calls:

- Get(): Returns the channel config and resource config
- Simulate(ConfigEnvelope): Simulates a config, and reports
  whether it succeeded or failed.

The change set includes:
- Proto changes (that contribute to the size of the change set)
- A resource service implementation that interacts with the peer
  via a support interface
- Unit tests for the service, that have 100% code coverage.
- Implementation of the support interface in the peer.
- Registration of the service to the gRPC server that services
  the endorser service (in other words, default port 7051)
