In block store ledger, there are 3 indexes that serve queries
of type ***ByTxid. These indexes maintain a mapping of txid to
(1) transactionEnv Location in block file,
(2) block location in block file, and
(3) the validation code respectively

This CR simplifies the code by merging above
three in a single index entry
