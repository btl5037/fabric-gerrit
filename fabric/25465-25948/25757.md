Test scenario has 2 orgs, which exercise the follwing cases, operating

-) Verify: Org1 lists orgs in sb EP
-) Org1 sets the value (succeeds)
-) Verify: Org1 reads the value
-) Org2 sets the value (fails)

-) Verify: Org1 lists orgs in sb EP
-) Org2 sets the value (fails)
-) Verify: Org1 reads the value
-) Org1+Org2 set the value (succeeds)
-) Verify: Org1 reads the value

-) Verify: Org1 lists orgs in sb EP
-) Org1+Org2 remove Org1 from the sb EP (succeeds)
-) Verify: Org1 lists orgs in sb EP
-) Org2 sets the value via cc2cc (succeeds)
-) Verify: Org1 reads the value
-) Org1 sets the value via cc2cc (fails)
-) Verify: Org1 reads the value
