This CR implements a simple lru caching library
for the ledger.

(1) cache is bounded by the size rather than number
of entries.
(2) for each chain, there is a separate sub-cache
and a RWMutex to reduce the lock contention.
(3) fairness in utilized memory is provided among
chain caches.
(4) currently, lru eviction policy is supported. The
code can be easily extended to support lfu, arc, etc...
