1. Debug Message

pb.ChaincodeMessage_GET_STATE
-> "[%s] Sending %s", shorttxid(msg.Txid),

2. Error Message
   (sending ChaincodeMessage failed)

(1) "[%s] error sending %s",
    shorttxid(msg.Txid), pb.ChaincodeMessage_XXXXXX
(2) "[%s] error sending XXXXXX", shorttxid(msg.Txid)
(3) "[%s] error sending XXXXXX %s",
    shorttxid(msg.Txid), pb.ChaincodeMessage_XXXXXX
(XXXXXX is a ChaincodeMessage type)
The same format should be used.
