The operations in the read write of a transactions are executed
at the validate/commit time. The LockBasedTxSimulator has been
modified to simulate the reads/writes performend by an operation.
The simulator constructed for operations has "read your
own write" and "read writes of the previous transactions in the
same block" semantics. The state based validator get the simulator
results and adds the writes to the write set of the transaction
for each operation in the transaction.
