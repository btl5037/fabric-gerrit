This CR changes the funcitons in the interface DeployedChaincodeInfoProvider
such that they take an additional parameter for the channelName.
The main motivation for passing this additional param is to allow
an implementation to construct the org-implicit collections on-the-fly
(by using the channel config) and not mandatorily persisting the details
